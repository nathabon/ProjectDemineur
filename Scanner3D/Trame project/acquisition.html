    import cv2
import time

cap = cv2.VideoCapture(0)
time.sleep(3)   # Attente focus caméra

angleObjet = 0

while (angleObjet < 360):   # Boucle principale

    _, frame = cap.read()
    cv2.imshow('frame', frame)

    angleObjet = angleObjet + 1
    print(angleObjet)

    if cv2.waitKey(1) == ord('q'):  # Un appui sur la touche q permet de quitter l’application
        break

cv2.destroyAllWindows()
cap.release()

    La boucle principale while permet de simuler une mise en rotation de l'objet sur 360°.

    import cv2
import time

cap = cv2.VideoCapture(0)
time.sleep(3)   # Attente focus caméra

_, frame = cap.read()

hauteur_image, largeur_image, canaux_image = frame.shape

print("L'image comporte", hauteur_image, "pixels en hauteur et", largeur_image, "pixels en largeur")

frame.itemset((8, 9, 2), 250)
frame.itemset((8, 9, 1), 255)
frame.itemset((8, 9, 0), 255)

if frame.item(8, 9, 2) == 255 and frame.item(8, 9, 1) == 255 and frame.item(8, 9, 0) == 255:
    print("Le pixel (8,9) est blanc")

cv2.imshow('frame', frame)

cv2.destroyAllWindows()
cap.release()

    L’objet frame est représentatif de l’image capturée par la caméra.

    La variable hauteur_image contient la hauteur de l’image.

    La variable largeur_image contient la largeur de l’image.

    La ligne cv2.imshow('frame', frame) permet d’afficher l’image capturée.

    Chaque pixel de l’image est codé par trois octets représentant le niveau de rouge (sous-pixel rouge), vert (sous-pixel vert) et bleu (sous-pixel bleu) du pixel.

    Le noir est codé en décimal de la façon suivante : 0 0 0

    Le blanc est codé en décimal de la façon suivante : 255 255 255

    En Python, pour mettre en noir un pixel situé à la ligne 8 et la colonne 9 on écrit :

    1
2
3
    framecouranteRGB.itemset((8, 9, 2), 0)      # Niveau de rouge à 0
framecouranteRGB.itemset((8, 9, 1), 0)      # Niveau de vert à 0
framecouranteRGB.itemset((8, 9, 0), 0)      # Niveau de bleu à 0

    En Python, pour mettre en blanc un pixel situé à la ligne 8 et la colonne 9 on écrit :

    1
2
3
    frame.itemset((8, 9, 2), 255)    # Niveau de rouge à 255
frame.itemset((8, 9, 1), 255)    # Niveau de vert à 255
frame.itemset((8, 9, 0), 255)    # Niveau de bleu à 255

    Pour tester si un pixel est d'une couleur donnée, par exemple tester si le pixel de la ligne 8 et de la colonne 9 est blanc, on va simplement tester si le sous-pixel rouge est à 255 en écrivant la condition suivante :

    1
2
3
    frame.item(8, 9, 2) == 255
frame.item(8, 9, 1) == 255
frame.item(8, 9, 0) == 255
