    Phase 5 : Génération du fichier

    Objectif de cette phase

    L’objectif est ici de mettre en place la création puis la mise à jour d'un fichier, que l'on nommera fichier.ply, après chaque calcul de coordonnées 3D des points présents dans la ligne de pixels blancs désépaissie.

    Explications

    On souhaite ici que le fichier fichier.ply soit compatible avec le standard PLY.

    Ce fichier décrira un nuage de points dans un espace 3D.

    Ce fichier devra être créé dès le début de l'exécution du code Python puis être mis progressivement à jour au fur et à mesure des calculs.

    Méthodes permettant de manipuler un fichier texte

    Le fichier PLY est un fichier texte dont l'extension est .ply.

    Il sera nécessaire d'utiliser les méthodes permettant de manipuler les fichiers de type texte.
